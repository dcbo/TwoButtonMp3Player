# TwoButtonMp3Player
MP3-Player which plays only one song.

Controlled with two Buttons:
Green Button: Start Playing Song
Red Button :  pause / resume 

# Purpose
Toy for my daughter which play the toothbrush-Song

# Hardware
- Arduino Nano
- DF-Player
- Speaker
- 2 Buttons
